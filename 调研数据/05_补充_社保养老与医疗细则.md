## 四、养老金与企业退休待遇

### 4.1 养老金调整政策（2025年）

- 人力资源社会保障部 财政部《关于2025年调整退休人员基本养老金的通知》（人社部发〔2025〕38号，2025年7月7日发布）
- **总体调整水平**：2024年月人均基本养老金的 **2%**（上一年为3%）
- **调整办法**：定额调整 + 挂钩调整 + 倾斜调整（向养老金较低群体、高龄、艰苦边远地区倾斜）

### 4.2 全国企业退休人员月均养老金估算

| 年份 | 月均养老金（元） | 涨幅 |
|------|:--------------:|:----:|
| 2023年 | 3,162（官方数据） | — |
| 2024年 | **≈3,257**（按+3%推算） | +3% |
| **2025年** | **≈3,322**（按+2%推算） | **+2%** |

### 4.3 分地区差异

| 地区 | 约月均企业养老金水平 |
|------|:------------------:|
| 上海 | **~6,200 元** |
| 北京 | **~5,800 元** |
| 广东 | ~4,100 元 |
| 全国平均 | ~3,322 元 |
| 黑龙江/中西部 | 2,400 ~ 3,200 元 |
| 城乡居民基础养老金 | **~205 元/月** |

### 4.4 养老金替代率（需关注）

| 类别 | 替代率 |
|------|--------|
| **企业职工养老金替代率** | **约 40-45%**（退休前工资的40-45%） |
| 国际警戒线 | 55%（低于此线退休生活质量明显下降） |
| 机关事业单位 | 约 70-90% |

### 5.3 当前代码中的 RETIREMENT 数据校验

代码中已有：
```python
"care_monthly": {
    "居家养老（基本生活费）": {"一线":3000, ..., "五线":1300},
    "普惠养老机构": {"一线":6000, ..., "五线":2000},
    "中高端养老机构": {"一线":12000, ..., "五线":3500},
}
```
- **居家养老**：基本吻合，一线3,000偏低（建议3,500-4,000），三线1,800基本合理。
- **普惠养老机构**：一线4,000更典型（而非6,000），新一线3,000-5,000。
- **中高端养老机构**：一线12,000基本正确，三线5,000偏高（实际3,500-5,000）。
- **核心矛盾**：企业养老金3,322元 vs 普惠养老机构4,000-6,000元，存在明显缺口。
