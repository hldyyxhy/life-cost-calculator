# 调研数据索引

本目录存放「生活成本计算器」的全部底层数据调研成果，供程序数据模型编写时参照。
所有数据基于 2023-2026 年公开来源，已附来源链接；查不到精确值的项均标注了「估算」。

## 文件清单

| 文件 | 内容 | 用途 |
|---|---|---|
| `01_城市分级体系.md` | 一/新一线/二/三/四/五线城市的划分、代表城市名单、各级房价/收入/相对成本系数 | 两个计算器的「城市等级」维度基础 |
| `02_从生到死生命周期成本.md` | 怀孕→0~22岁逐年养育→婚房/结婚→社保个税→养老→丧葬，分普惠/中产/高端三档 | **计算器1：从生到死成本** |
| `03_当前生存成本.md` | 各线城市成年人月度生存成本（住房/饮食/交通/通讯/日用/社保）+ 收入端（工资/最低工资/个税） | **计算器2：当前生存境况** |
| `04_补充_宏观经济与收入数据.md` | 2025年全国经济总量、居民收入五等分、各线城市收入估算、人均消费支出八大类细分 | 两个计算器的收入端校验 |
| `05_补充_社保养老与医疗细则.md` | 各城市社保缴费基数、灵活就业缴费、养老金水平、养老院收费、医疗支出、丧葬费用 | **补充二/三中社保/医疗/养老/丧葬数据** |
| `06_补充_房价房租与通勤数据.md` | 2025年百城房价排名、房价收入比、各城市详细房租、公积金/商贷利率、通勤成本 | **补充一/三中住房与交通数据** |

## 数据使用约定（写程序时遵循）

1. **城市系数**（以全国城镇平均=1.0）：一线 1.8-2.0、新一线 1.3-1.5、二线 1.1-1.2、三线 0.9-1.0、四线 0.8-0.9、五线 0.7-0.8。代码中实际取值以「三线=1.0」为内部基准（一线2.0、新一线1.5、二线1.2、三线1.0、四线0.82、五线0.70）。
2. **养育档位系数**（以普惠=1.0）：普惠 1.0、中产 2.2、高端 4.5（教育/月子类高端可达 8-10，单独处理）。
3. 彩礼、丧葬等强文化属性项目，**不套通用系数**，按城市/地域单独给值。
4. 所有金额默认单位：人民币元。区分「年度」「累计」「月度」时在 UI 标注。
5. UI 应明确标注「估算值」字样，避免误导。
6. **各线级数据更新频率**：房价/房租月更新，工资/收入/社保基数年更新（每年5-7月），生育成本报告每1-2年发布。建议每年复查更新数据模型。
