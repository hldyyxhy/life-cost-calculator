## v1.2 — 2026-06-21

### 新增功能
- **结果富文本化**：买 vs 租等结果区用富文本——标题/结论/警示分色，行内**数字加粗**、负面评级词（偏高/违法/失控等）标红，不再一片同字体看不清重点。
- **提示词填档案事实**：「问 AI」提示词自动带上档案里的年龄/性别/健康/社保/家庭/存款/负债等客观情况，AI 分析更充分（原则：只填确认的事实，不填工具估算——工具本身不准，塞进去反而误导）。
- **性别字段**：档案可填性别（4050 补贴等按性别判断的功能更准）。
- **处境页「按实际改」**：结果区的估算支出可就地改成你的实际数字（住房/饮食/交通/通讯/给老家/社保），改完重算结余/解读——默认估算降门槛，知情用户可改成实际值。
- **长期跟踪**：关闭程序时可存「跟踪快照」（填姓名，同名跨日期累积，自动生成 json + 文本两份）；新增「⑨ 长期跟踪」页，看月结余/存款/月度成本/月负债/结余率的变化趋势图 + 历次明细，可导出文本档案。让用户看到自己有没有在变好。

### 优化
- 城市与住房页顶部红字去掉「房价/租金涨幅是假设」（公式已按房价不变估算，不再矛盾）。

### 规划
- 新增《多端迁移方案.md》：确定小程序多端技术路线（Taro/uni-app + 一份 TS 核心 + 云端 JSON 数据，计算端侧离线/数据上云热更新），已论证排除 Flet（编译不出小程序）、云函数跑计算（微信云函数不收 Python + 违背离线）、FastAPI+React 网页版（要服务器）。

## v1.1 — 2026-06-20

### 新增功能
- **数据持久化**：打开即恢复上次档案 + 债务页输入，关闭即自动保存（程序同级 `data/`）。
- **打包 exe**：单文件双击即用，无需装 Python（PyInstaller，约 11MB）。
- **首次档案填写向导**：7 步分步问答 + 智能跳过。答题式依次填选，避免一整页表单带来的心理压力。
- **提示词一键打开 AI**：复制提示词 + 自动打开 AI 网页（默认 DeepSeek，可选豆包/Kimi）。
- **多子女区分年龄段**：不同年龄段各填数量，养娃成本/专项扣除按段分别累加。
- **反诈骗模块**：6 类骗局按钮墙 + 特征速查 + 紧急止损（96110/110/国家反诈 APP）。
- **导航优化**：可滚动 + 滚动条自动隐藏；求助与反诈合并为一页（导航 9→8 项）。
- **住房决策**：买 vs 租 / 公积金额度 / 利率压力测试（「城市与住房」页 Notebook 四标签）。
- **个税优化**：年终奖单独 vs 合并计税对比 + 专项附加扣除提醒。
- **本地救助对照**：223 城低保标准，符合/边缘/特困三层判断。
- **医保就医**：31 省住院报销估算（基本+大病+异地）+ 医保速查（药品目录/慢病/DRG）。
- **债务健康仪表盘**：负债率/月供比/还清月数/风险等级 + 失控（还款盖不住利息）检测。

### 修复
- 一线灵活就业社保偏低 1880→2050、一线最低工资 2600→2700（生存成本/社保公式微调，逻辑未动）。
- 城市对比 `compare_cities` 漏传子女段（多子女改造时一并修复）。
