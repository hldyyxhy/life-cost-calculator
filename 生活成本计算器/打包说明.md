# 打包说明 —— 生成「生活成本计算器.exe」

把程序打包成单个 exe，劳动者无需装 Python，双击即用。

## 一、环境要求

- Windows（产物是 Windows exe）
- Python 3.10+（开发用 3.13；装时勾选 tcl/tk 和 tkinter）
- PyInstaller：`python -m pip install pyinstaller`

## 二、打包步骤

在 `生活成本计算器/` 目录下执行：

```
pyinstaller build.spec --clean
```

或等价命令行（不依赖 .spec）：

```
pyinstaller --onefile --windowed --name "生活成本计算器" --clean main.py
```

等待 1~3 分钟，产物在：

```
dist/生活成本计算器.exe   ← 要分发的文件（约 11 MB）
```

中间文件在 `build/`，可随时删除。

## 三、打包后验证（重要）

1. 把 `dist/生活成本计算器.exe` 单独拷到一个**空文件夹**（模拟用户环境，确认不依赖开发目录）。
2. 双击运行：正常弹出主窗口 = 通过；闪退/无反应见下方常见问题。
3. 填档案点「确定」→ 关闭 → 再双击 → 档案应自动恢复（验证 `data/` 持久化在 exe 同级生效）。
4. 各功能页（处境/对比/三座山/借贷/权益）各点一次「算一算」，确认无报错。

## 四、分发时给用户什么

- 只需 `生活成本计算器.exe` 单个文件。
- 无需 Python、无需安装，U 盘拷走直接双击。
- 用户档案/债务数据存在 exe 旁的 `data/` 文件夹；**换电脑时把 exe + data/ 一起拷走可保留数据**。

## 五、常见问题

- **双击闪退 / 杀毒软件拦截**：onefile exe 会解压到临时目录运行，部分杀毒软件（如 360）可能误报。
  让用户加白名单；或改 onedir 模式（`pyinstaller --windowed --name "生活成本计算器" main.py`，去掉 `--onefile`，产物是文件夹，启动更快、误报更少）。
- **首次启动慢（几秒）**：onefile 要解压到临时目录，正常；系统缓存后下次会快些。
- **中文 / 特殊路径**：exe 偶尔在含特殊字符的路径下出问题，建议放简单路径。
- **Windows SmartScreen 拦截**：未签名 exe 首次运行会被拦，点「仍要运行」；正式分发需代码签名证书（成本较高，个人项目通常不签）。

## 六、更新数据后重新打包

`rights_data.py` / `cost_data.py` 等数据更新后，重跑第二步的 pyinstaller 命令即可，`dist/` 自动覆盖。
数据已编译进 exe，用户换上新 exe 即拿到最新数据。

## 七、build.spec 说明

- `--onefile`：单 exe，分发友好。
- `console=False`：GUI 程序不弹黑色控制台窗口（即 `--windowed`）。
- `datas=[]`：本程序零外部资源（无图片/字体/json 模板），无需打包资源；`pages` 子包由 import 链自动收集。
- `icon=None`：暂无图标；备好 `app.ico` 后改 `icon='app.ico'` 再重新打包。

## 八、.gitignore

`build/`、`dist/`、`*.exe` 已加入 `.gitignore`，不入库（产物可随时重新生成）。
