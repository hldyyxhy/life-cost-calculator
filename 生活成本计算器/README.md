# 生活成本计算器

> 帮劳动者看清钱去哪了、换城市值不值、三座大山（结婚/养娃/养老）要多少钱。

## 这是什么

一个面向无产阶级和广大劳动者的桌面工具，包含三个核心模块：

1. **我现在的处境（输入计算）**——输入年龄、工资、城市、生活方式、家庭负担等，
   计算你当前每月的生存成本、到手收入（扣五险一金+个税）与结余/缺口。
   新增：工资去向堆叠条、城市生存底线、结余率健康评等、攒首付年限预测。

2. **城市加减法（对比计算）**——选择当前城市和目标城市，对比生活质量指标，
   回答"换城市值不值"的核心问题。自动预估目标城市工资，并排显示六项指标。

3. **人生三座山（独立计算）**——三个独立模块：结婚要花多少钱、养娃要多少钱、
   养老有多大缺口。各自独立，不需要一次性填完。

## 如何运行

### 前提
已安装 Python 3。**无需任何第三方包**，仅用标准库 tkinter。

### 运行方式（任选其一）
- **最简单**：双击 `main.py`
- **命令行**：`python main.py`

## 数据来源与可靠性

- 所有数据来自 2023-2026 年公开调研报告（育娲《中国生育成本报告》、国家统计局、
  北大 CIEFR、国家医保局、人社部、各市统计局等）。
- 完整调研底稿见 `调研数据/` 目录。
- **所有数字均为估算中值**，个体差异可能很大，仅供了解量级与结构，
  **不作为任何理财、投资或消费决策的依据**。

## 计算模型说明

采用「基准值 × 城市系数 × 养育档位」模型：
- 城市系数以**三线城市≈全国城镇平均 = 1.0** 为基准（一线2.0、新一线1.5、二线1.2、
  三线1.0、四线0.82、五线0.70）。
- 养育档位系数：普惠1.0、中产2.2、高端4.5（教育类、月子类高端额外加权）。
- 彩礼、丧葬等强文化属性项目不套系数，按城市/地域单独取值。
- 个税按 5000 元起征点 + 七级累进 + 专项附加扣除计算。
